## 🌐 Personal Portfolio Website (First Version)

**Project Description (Resume Paragraph):**
Developed a static personal portfolio website to present academic projects, technical skills, and contact information through a structured and user-friendly web interface. Designed multiple sections including profile overview, project showcase, and navigation components to improve professional visibility and online presence. Implemented responsive layout techniques and basic interactive elements to ensure accessibility across different screen sizes and devices. This project helped build foundational experience in frontend web development, website structuring, and deployment workflow.

**Technologies Used:**

* HTML
* CSS
* JavaScript
* Responsive Web Design
* Static Web Hosting (GitHub Pages / Netlify)
